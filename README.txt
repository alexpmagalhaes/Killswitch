README

System requirements:

pandas >= 1.0
seaborn
matplotlib
numpy
Json

Installation guide:

Install dependencies via conda or PIP.

Instructions for use:

Open Jupiter or Google Colab and run notebooks by order:

00-AF3_score_visualizer.ipynb outputs alphafold 3 contact probability and PAE matrix.
01-Diffeerences_matrix.ipynb calculates subtraction matrix and outputs heatmaps.

Expected output:

contact_data_0.csv (raw data used for the contact probability heatmap)pae_heatmap_0.csv (raw data used for the PAE matrix)
Heatmaps that can be saved as PNG or PDF

Expected run time: 60s
